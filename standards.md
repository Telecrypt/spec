| Feature                         | Standard                                       | Description                                                                                     |
| ------------------------------- | ---------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| Ed25519 Public Key Cryptography | [RFC8032](https://tools.ietf.org/html/rfc8032) | High security level, battle-tested, audited implementations in relevant languages to Telecrypt. |
